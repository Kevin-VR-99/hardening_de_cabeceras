Demostración de HttpOnly

Cómo protege una cookie contra acceso desde JavaScript

¿Qué es HttpOnly?

HttpOnly es un atributo de cookie que impide que JavaScript del navegador acceda a esa cookie mediante document.cookie.

Esto ayuda a reducir el riesgo de robo de sesión en ataques XSS.

Aquí la cookie se crea sin HttpOnly. Por eso, JavaScript sí puede leerla usando document.cookie.

¿Qué debes mostrar en tu demostración?

  • En el caso vulnerable, el navegador sí mostrará la cookie en pantalla.
  • En el caso protegido, document.cookie no mostrará la cookie HttpOnly.
  • En DevTools → Application → Cookies podrás ver que existe.
  • Pero en consola JavaScript no será accesible.
